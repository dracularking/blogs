这是第一篇文章。

当前博客方案：

- Hugo 负责生成静态站点。
- PaperMod 负责主题和基础博客体验。
- GitHub Pages 免费托管静态文件。
- Cloudflare 管理域名 DNS、HTTPS 和缓存。
- Giscus 基于 GitHub Discussions 提供评论。

部署后，把 `hugo.yaml` 中的站点标题、作者、GitHub 地址和 Giscus 参数替换成真实值即可。
